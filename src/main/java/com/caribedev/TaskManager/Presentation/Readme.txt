Capa Presentation
En esta capa colocaremos las clases que representan a los objetos encargados de 
la interacción con el usuario y la exposición de los datos procesados.
Su objetivo principal es actuar como la interfaz entre el sistema y el usuario, 
asegurando que la información fluya correctamente desde la capa de aplicación 
hacia la capa de presentación.
Se relaciona con la Application Layer, que proporciona los casos de uso y la 
lógica de negocio. También se conecta con las subcapas internas para gestionar 
adecuadamente las respuestas y formatos de datos.
