Controllers
En esta subcapa colocaremos las clases que manejan las solicitudes de los usuarios 
y coordinan la ejecución de los casos de uso de la capa de aplicación.
Su objetivo principal es recibir, interpretar y procesar las peticiones, enviando 
las respuestas adecuadas.
Se relaciona con la subcapa Dtos para gestionar los datos de entrada y salida, 
así como con Exceptions para manejar posibles errores.
